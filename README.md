# ExploreEase Travels

## Project Overview

ExploreEase Travels is a responsive travel website developed as part of the **MANTRA 2026 Summer School Frontend Website Assignment**. The website provides information about travel destinations, tour packages, a gallery, and an online booking/contact form.

---

## Website Pages

- Home
- About
- Packages
- Gallery
- Contact

---

## Features

- Responsive design for Mobile, Tablet, and Desktop
- Easy navigation menu
- Beautiful Hero Banner
- Popular Travel Destinations
- Travel Packages
- Image Gallery
- Contact & Booking Form
- Form Validation using JavaScript
- Basic SEO implementation
- Clean and modern UI
